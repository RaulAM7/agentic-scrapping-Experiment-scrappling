# Reports

Esta carpeta alojara reportes de analisis del caladero.

Reportes posibles:

- source assessment report;
- sample evaluation report;
- top opportunities report;
- topic distribution report;
- geography distribution report;
- entity type report;
- scraping quality report.

Los reportes de esta carpeta sirven para priorizar y explicar el caladero. No sustituyen a una capa canonica de leads con contactabilidad cerrada.
