# GLOSSARY

- **ICP (Ideal Customer Profile)**: Strategic definition of the target customer segment — here, document-heavy professional firms in the Canary Islands.

- **Despacho**: Spanish term for a professional office/firm (law, accounting, tax advisory). The primary unit of the target ICP.

- **Scrapling**: Python library for web scraping and data extraction; the stack under evaluation in this repo.

- **Agentic workflow**: A repository-based, AI-assisted, structured workflow where an LLM-driven operator executes tasks within a defined system of specs, context, and deliverables.

- **Inversion del sujeto pasivo**: Reverse charge VAT mechanism applicable to transactions from mainland Spain (Peninsula) to the Canary Islands. A critical fiscal rule in the target ICP's domain.

- **Smart scraping**: Web data extraction that goes beyond raw crawling — includes structured extraction, signal collection, categorization, and discovery, oriented toward business research rather than pure data harvesting.
